# 엠팍커(MLBParker)

엠팍커(MLBParker)는 엠엘비파크(MLBPark) 게시글 작성자의 정보를 Chrome Side Panel에서 확인하고, 해당 작성자의 최근 활동 내역을 빠르게 살펴볼 수 있게 해주는 크롬 확장 프로그램입니다.

게시글을 읽다가 작성자가 어떤 글을 주로 쓰는지, 어느 게시판에서 활동하는지, 어떤 말머리를 자주 사용하는지 궁금할 때 페이지를 벗어나지 않고 바로 확인할 수 있습니다.

## 한눈에 보기

- 게시글 보기 페이지에서 작성자 정보를 자동 감지합니다.
- 사이드 패널에서 가입일, 최근 게시글, 태그 통계를 바로 확인할 수 있습니다.
- 게시판 필터와 태그 필터로 관심 있는 글만 빠르게 좁혀볼 수 있습니다.
- 탭별 상태를 기억해 여러 글을 오갈 때도 흐름이 끊기지 않습니다.
- 차단 기능을 통해 특정 작성자를 바로 차단할 수 있습니다.

## 주요 기능

- 게시글 보기 페이지에서 작성자 UID, 닉네임, 프로필 이미지를 자동 감지
- Chrome Side Panel에서 작성자 정보 즉시 표시
- 작성자의 최근 게시글 목록 조회
- 게시판 필터 지원: 전체, MLB타운, 한국야구타운, BULLPEN
- 말머리(태그) 통계 집계 및 태그 기반 필터링
- 작성자 가입일 조회
- 현재 보고 있는 게시글 하이라이트 표시
- 작성자 차단 버튼 제공
- 탭별 상태 캐시 및 복원
- 일부 태그 통계의 로컬 캐시 저장
- 우클릭 컨텍스트 메뉴로 사이드 패널 열기

## 동작 방식

1. 사용자가 MLBPark 게시글 보기 페이지를 엽니다.
2. `content.js`가 페이지에서 작성자 UID, 닉네임, 사이트 정보, 게시글 ID를 추출합니다.
3. `background.js`가 탭 단위로 작성자 정보를 캐시하고, 현재 활성 탭 기준으로 사이드 패널 상태를 동기화합니다.
4. `sidepanel.js`가 작성자의 가입일, 게시글 목록, 태그 통계를 불러와 렌더링합니다.
5. 사용자는 게시판 필터, 태그 필터, 더보기, 차단 기능을 사이드 패널에서 바로 사용할 수 있습니다.

## 설치 방법

1. 이 저장소를 내려받거나 압축을 풉니다.
2. Chrome에서 `chrome://extensions` 로 이동합니다.
3. 우측 상단의 `개발자 모드`를 켭니다.
4. `압축해제된 확장 프로그램을 로드합니다`를 클릭합니다.
5. 이 프로젝트 폴더를 선택합니다.

설치 후에는 확장 프로그램 아이콘을 클릭하거나, 페이지에서 우클릭 후 컨텍스트 메뉴를 통해 사이드 패널을 열 수 있습니다.

## 사용 방법

1. `https://mlbpark.donga.com/` 의 게시판에서 게시글 보기 페이지를 엽니다.
2. 확장 프로그램이 작성자 정보를 감지하면 우측 사이드 패널에 해당 작성자의 정보가 표시됩니다.
3. 사이드 패널에서 최근 게시글 목록, 가입일, 말머리 통계를 확인합니다.
4. 필요하면 게시판 필터 또는 태그 필터를 적용해 관심 있는 글만 좁혀 봅니다.
5. `더보기` 버튼으로 다음 글 목록을 추가로 불러옵니다.
6. 필요 시 `차단` 버튼으로 작성자를 차단합니다.

## 지원 페이지

- MLBPark 게시글 보기 페이지
- 매니페스트상 콘텐츠 스크립트 대상
- `https://mlbpark.donga.com/mp/b.php?*`
- `https://mlbpark.donga.com/mp`
- `https://mlbpark.donga.com/mp/`

실제 작성자 추출은 URL 쿼리에 `id`가 있는 게시글 보기 페이지를 기준으로 동작합니다.

## 권한 설명

`manifest.json`에서 아래 권한을 사용합니다.

- `sidePanel`
  - 작성자 정보와 게시글 목록을 전용 사이드 패널에 표시하기 위해 사용합니다.
- `tabs`
  - 활성 탭 변경, 탭 이동, 탭별 캐시 복원을 처리하기 위해 사용합니다.
- `storage`
  - 현재 작성자 정보와 태그 통계 캐시를 저장하기 위해 사용합니다.
- `contextMenus`
  - 우클릭 메뉴에서 사이드 패널을 열기 위해 사용합니다.
- `declarativeNetRequest`, `declarativeNetRequestWithHostAccess`
  - MLBPark 리소스를 사이드 패널 환경에서 안정적으로 표시하기 위해 응답 헤더 일부를 조정하는 데 사용합니다.
- `host_permissions: https://mlbpark.donga.com/*`
  - MLBPark 페이지의 작성자 정보, 게시글 목록, 가입일 조회를 위해 사용합니다.

## 파일 구조

```text
.
├── manifest.json
├── background.js
├── content.js
├── sidepanel.html
├── sidepanel.js
├── icons/
└── README.md
```

- `manifest.json`
  - 확장 프로그램 설정, 권한, 콘텐츠 스크립트, 사이드 패널 진입점 정의
- `background.js`
  - 탭 상태 관리, 메시지 중계, 사이드 패널 동기화, 컨텍스트 메뉴 등록
- `content.js`
  - MLBPark 게시글 페이지에서 작성자 정보 추출
- `sidepanel.html`
  - 사이드 패널 UI 마크업
- `sidepanel.js`
  - 작성자 프로필, 가입일, 게시글 목록, 태그 통계 렌더링 및 필터 처리

## 개발 메모

- Manifest V3 기반입니다.
- 별도 빌드 과정 없이 정적 파일만으로 동작합니다.
- 외부 패키지 매니저 의존성(`package.json`)은 없습니다.
- 태그 통계는 일부 결과를 로컬 캐시에 저장해 재조회 비용을 줄입니다.
- 게시글 목록 조회와 가입일 조회는 MLBPark 응답 구조에 의존하므로, 사이트 마크업 변경 시 수정이 필요할 수 있습니다.

## 알려진 제약

- 게시글 보기 페이지가 아닌 목록/메인 화면에서는 작성자 정보가 표시되지 않을 수 있습니다.
- MLBPark DOM 구조나 검색 결과 구조가 바뀌면 작성자 추출 또는 목록 파싱이 깨질 수 있습니다.
- 태그 통계는 여러 페이지를 순회해 계산하지만, 사용자의 전체 이력을 완전하게 보장하는 방식은 아닙니다.
- 차단 기능은 MLBPark 측 요청 성공 여부에 의존합니다.
