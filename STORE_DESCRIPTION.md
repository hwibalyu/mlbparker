# 엠팍커(MLBParker) - 스토어 설명

## 패키지 요약
엠팍(MLBPark) 게시글 작성자의 정보를 사이드 패널에서 간편하게 확인하고 게시글 목록을 스캔합니다.

## 설명
엠팍커(MLBParker) - 엠엘비파크 유저 분석 사이드 패널
"엠팍 이용이 더 스마트해집니다! 이제 페이지 이동 없이 작성자의 히스토리를 한눈에 확인하세요."

엠팍커(MLBParker)는 국내 최대 커뮤니티 중 하나인 엠엘비파크(MLBPARK) 이용자들을 위한 크롬 익스텐션입니다. 게시글을 읽다가 작성자가 궁금할 때, 일일이 클릭해서 새 창을 띄울 필요가 없습니다.

### 1. 주요 기능
- **실시간 사이드 패널 표시**: 엠팍 내 작성된 글을 클릭하면, 브라우저 우측 사이드 패널에 해당 유저의 정보가 즉시 나타납니다.
- **게시글 목록 스캔**: 유저가 이전에 작성했던 글 목록을 한 번에 스캔하여 리스트로 보여줍니다.
- **차단 기능**: 차단 버튼을 통해서 해당 유저의 글을 차단할 수 있습니다.
- **끊김 없는 브라우징**: 보고 있던 게시글을 떠나지 않고도 작성자의 성향이나 활동 내역을 파악할 수 있어 효율적입니다.
- **직관적인 UI**: 엠팍의 고유 컬러감을 살린 깔끔한 디자인으로 가독성을 높였습니다.

### 2. 사용 방법
1. 엠엘비파크 게시판에서 궁금한 작성자의 정보를 확인합니다.
2. 익스텐션을 실행하거나 지정된 기능을 통해 유저를 검색합니다.
3. 오른쪽 사이드 패널에 나타나는 글 목록과 정보를 확인하며 커뮤니티를 즐깁니다!
